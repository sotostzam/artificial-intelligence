# Uniform Cost search

## Time and space complexity

* *"b"* represents the branching factor
* *"C"* represents the cost of the optimal solution
* *"e"* represents minimum cost of a step

|  Time Complexity  | Space Complexity |
| :---------------: | :--------------: |
|    O(b^(C/e))     |    O(b^(C/e))    |
